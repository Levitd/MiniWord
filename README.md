# MiniWord

**MiniWord** — лёгкий бесплатный текстовый редактор для Windows с нативной поддержкой формата **.docx** (совместим с Microsoft Word). Осознанно минимальный набор функций и понятный интерфейс в стиле Word, без перегруженности и подписок — альтернатива тяжёлым офисным пакетам, когда нужно просто открыть, отредактировать, распечатать и сохранить документ. Установщик весит около 3 МБ, само приложение запускается мгновенно.

Интерфейс полностью переведён на **русский и английский** (язык переключается в настройках и применяется сразу). Приложение умеет работать **с несколькими документами одновременно** — каждый в своём окне, — запоминает **последние открытые файлы** и **размер/положение окна**, а также **само проверяет обновления** и умеет их скачивать и устанавливать.

*MiniWord is a lightweight, free Windows word processor with native **.docx** support (Microsoft Word compatible). A deliberately minimal, Word-like editor without bloat or subscriptions — a fast alternative to heavy office suites for everyday documents. Fully localized in Russian and English, works with multiple documents at once, remembers recent files and window size, and updates itself.*

## Возможности / Features

**Форматирование текста**
- 📄 Открытие и сохранение **.docx** (DocumentFormat.OpenXml — официальный SDK Microsoft)
- 🔤 Выбор шрифта и размера (в пунктах, как в Word), **жирный** / *курсив* / подчёркнутый
- 🎨 Цвет текста и заливка (выделение маркером), палитры в стиле Word
- 📐 Выравнивание, отступы (в т.ч. первой строки), межстрочные и межабзацные интервалы
- 📝 Маркированные и нумерованные списки
- 🖼️ Вставка картинок (сохраняются внутри .docx)

**Страница и печать**
- 📃 Вид страницы «белый лист», размеры A4 / Letter / A5, границы страниц в редакторе
- 🔢 Колонтитулы и нумерация страниц — сохраняются в .docx как настоящие header/footer с полем PAGE; есть галочка «колонтитул на первой странице» (режим «другая первая страница»)
- 🖨️ Печать и постраничный предварительный просмотр

**Работа с файлами**
- 🗂️ **Несколько документов одновременно**: открытие нескольких файлов сразу и отдельное «Новое окно» (Ctrl+Shift+N)
- 🕘 **Последние 5 файлов**: список в меню «Файл» и стартовый экран при запуске без документа
- 🔗 Открытие через «Открыть с помощью…» из проводника Windows
- ⚙️ Настройки: шрифт и размер по умолчанию, язык интерфейса, размер страницы; окно запоминает свой размер и положение

**Прочее**
- ↩️ Undo / Redo и горячие клавиши (Ctrl+N/O/S/P, Ctrl+B/I/U)
- 🌍 Интерфейс на **русском и английском** с мгновенным переключением
- 🔄 **Автообновление**: приложение проверяет наличие новой версии на GitHub и предлагает обновиться — сразу, **в фоне** (установить при закрытии или следующем запуске), или позже. Без сети проверка тихо пропускается и не мешает работе.
- ℹ️ «О программе»: версия, дата релиза, автор

## Скачать / Download

**[⬇ Скачать установщик (Releases)](https://github.com/Levitd/MiniWord/releases/latest)** — `MiniWordSetup-x.x.x.exe` (~3 МБ)

Установщик на русском и английском: выбор папки установки, ярлыки на рабочем столе и в меню Пуск (из меню Пуск можно закрепить на панели задач: ПКМ → «Закрепить на панели задач»).

Требуется [.NET 6 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/6.0/runtime) (или новее) — если он не установлен, установщик предложит открыть страницу загрузки.

*Requires .NET 6 Desktop Runtime or newer — the installer checks for it and offers the download page if missing.*

## Сборка / Build

Требуется [.NET 6 SDK](https://dotnet.microsoft.com/download/dotnet/6.0) (или новее) на Windows.

```bash
cd MiniWord
dotnet build
dotnet run
```

Портативная сборка одним exe:

```bash
dotnet publish -c Release -r win-x64 --self-contained -p:PublishSingleFile=true
```

## Технологии / Tech stack

- **C# / WPF** (.NET 6, `net6.0-windows`)
- `RichTextBox` + `FlowDocument` — модель документа и редактирование
- **DocumentFormat.OpenXml** — чтение/запись WordprocessingML (.docx)
- WPF `DocumentPaginator` + XPS — печать и предпросмотр

## Ограничения / Limitations

Сознательно вне рамок проекта: таблицы, стили, сноски, рецензирование, совместное редактирование. Разбивка на страницы в окне редактора — приближённая (точная — в предпросмотре печати). Несколько документов открываются в отдельных окнах одного процесса.
